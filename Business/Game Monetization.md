# 9 Formas para se monetizar um game
1. Venda direta: Colocar um game a venda em uma loja física ou virtual (Steam, Play Store...);
2. Advergame: A construção de um jogo para uma marca, a monetização vem do contratante;
 - In-game advertising: Recursos dentro do jogo destinados para a promoção de uma marca (Outdoors, cartazes... dentro do jogo);
 - Product placement: Uso de um produto no contexto do jogo (Redbull para fornecer energia ao personagem...)
 3.  Around-game advertising: Banners de publicidade na HUD

4. Try before buy / Freemium: Jogo disponibiliza uma "amostra grátis";
5. DLC's e venda de produtos físicos: Venda de pacotes de expansão com novo conteúdo ou de merchandise;
6. Kickstart: Venda de projetos em sites de kickstart para estimular a chegada de investidores;
7. Microtransações: Venda de recursos virtuais ou gacha dentro do jogo;
8. Freeware: Jogo completamente grátis com funcionalidades para aceitar doações ou shares para promover a produtora;
