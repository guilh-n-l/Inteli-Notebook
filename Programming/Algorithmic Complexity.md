# Notação Big O, Assintótica ou Landau
> Caso um algorítimo execute um cálculo em cada item dentro de um array de tamanho $n$, o tempo em que esse algorítmo é executado é $O(n)$ (levando em consideração que o tempo de execução de cada elemento do array é $O(1)$)

Ex.: Caso um algorítmo demore o dobro do tempo necessário para analisar todos os elementos do array, ele é considerado um algorítimo $O(2n)$
