# Sistemas equivalentes
> 2 sistemas lineares são considerados equivalentes se possuem o conjunto solução;

## Checando se 2 sistemas são equivalentes
1. Mudar a ordem das equações (Trocar linhas);
2. Multiplicar uma das linhas de equação por um fator real ≠ 0;
3. Subtrair ou somar linhas;
4. Olhar se ambos os sistemas são identicos;
$$
\large{
\begin{matrix}
x_1 + x_2 + x_3 = 3 \\
-2x_1 - x_2 + 4x_3 = 1
\end{matrix}
\mbox{  and  }
\begin{matrix}
-2x_1 - x_2 + 4x_3 = 1 \\
2x_1 + 2x_2 + 2x_3 = 6
\end{matrix}}
$$
Ex.: Ao multiplicar a primeira linha do sistema da esquerda por 2, e trocá-la pela segunda, ambos os sistema ficam identicos, provando que são equivalentes;

# Escalonando sistemas
## Sistemas estritamente triangulares
- Um sistema se torna estritamente triangular quando o primeiro termo de cada linha da equação, com excessão da primeira linha, é o elemento $x_{k+1}$, considerando o primeiro elemento da linha imediatamente acima $x_k$;
$$
\large{
\begin{matrix}
x_1 + x_2 + x_3 = 3 \\
-2x_1 - x_2 + 4x_3 = 1
\end{matrix}
\mbox{Sistema não triangular}}
$$
$$
\large{
\begin{matrix}
-2x_1 - x_2 + 4x_3 = 1 \\
2x_2 + 2x_3 = 6 \\
5x_3 = 13
\end{matrix}
\mbox{Sistema estritamente triangular}}
$$
> OBS.: Para escalonar um sistema, é necessário utilizar as técnicas de criação de sistemas equivalentes;
