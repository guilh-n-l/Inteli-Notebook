# O que é?
> Encadeamento de estruturas simples (Blocos lógicos) com o objetivo de facilitar a escrita e a depuração;

# Tipos de estrutura
1. Sequência: Mostra cada bloco, ordenando-os em sequência de execução;
2. Seleção: Permite que o programa faça escolhas, escolhendo caminhos de execução;
3. Iteração: Permite a repetição de comandos;
