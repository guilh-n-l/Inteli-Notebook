# Blue-Green Deployment

> Técnica que transfere gradualmente o tráfego de usuários para diferentes versões de uma aplicação. o ambiente **blue** consiste na versão mais antiga de uma aplicação, enquanto o **green** consiste na versão mais nova. Assim, os desenvolvedores podem implementar novos recursos sem precisar desligar a aplicação como um todo;
